# **App Name**: Daily Planner Pro

## Core Features:

- Initial Setup: Users input their name, select subjects, and specify days for each subject. GPS and notification permissions are requested.
- Daily Schedule Display: Displays subjects for the next day based on the user's schedule.
- GPS Timezone Detection: Automatically detects the user's timezone via GPS, with manual fallback.
- Manual Date/Time Input: Provides a popup for manual date/time input if GPS is denied, to correctly calculate the next day's tasks.
- Task Details: Allows users to add details (description, multiple tasks) for each subject.
- Task Completion: Allows users to mark tasks as 'done', removing them from the list.
- Manual Task Addition: Enables users to manually add tasks with custom subjects, deadlines, and descriptions.

## Style Guidelines:

- Primary color: Soft purple (#A099FF) to evoke a sense of calm and focus.
- Background color: Light gray (#F0F0F5) for a clean, unobtrusive backdrop.
- Accent color: Muted teal (#73A7AD) for interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif that is modern while retaining warmth and readability.
- Use simple, consistent icons to represent subjects and actions.
- Clean, list-based layout for easy task viewing. Prioritize readability and intuitive navigation.
- Subtle transitions when marking tasks as complete and transitioning between days.
